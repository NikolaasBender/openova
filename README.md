# OpenOva

A lightweight VS Code clone built with Electron and Monaco Editor.

## Run on Linux

Follow these steps to get up and running in seconds.

### 1. Install Dependencies
```bash
npm install
```

### 2. Run in Development Mode
Starts the app with hot-reloading.
```bash
npm run dev
```

### 3. Build for Production
Creates a standalone executable structure.
```bash
npm run build
npx electron .
```
